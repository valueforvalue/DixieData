One Soldier record card, landscape orientation. Template: `soldier_landscape.typ`.

## Pre-iteration

The first PDF in this directory (`pre-iteration.pdf`) is the
source-of-truth baseline. Open it in your PDF viewer alongside this
file. Annotate by editing this file.

## How to annotate

For each issue, write a section with this shape:

```
### <short title>

- Term: <term name from TERMINOLOGY.md, by anchor #term-…>
- Location: <template:line or general area>
- What I see: <visible string or layout>
- What I want: <proposed visible string or layout>
- Why: <one sentence>
```

The agent reads this file, makes the corresponding code change,
regenerates the PDF, and reports back. Then you re-annotate and
we iterate.
